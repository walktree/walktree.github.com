libtorch yolov3

12月 4, 2018

Pytorch 1.0 预览版附带了一个好东西—libtorch，可以理解为 C++ 版本的 torch。和 pytorch 相比，依赖少，容易安装，非常适合产品化环境。

GitHub 上已经有很多版本的 YOLO v3 了，包括 darknet 的原版。不过，要在产品环境下面上面，还是没有一个满意的版本。

首先，产品环境下首选 C/C++，Pytorch 被排除了；其次，tensorflow 或 caffe 或许合适，有 C++ 的接口，不过团队（包括我）都接触不多。正好，Pytorch 前些日子发布了 1.0 预览版，多了 C++ 的支持。

由于是预览版，文档不全，实例代码基本没有，另外，对于 C++，算是初学者。整个过程不可谓不艰辛。代码写完，测试下来，发现检测结果不对，花了大量精力，回溯每一个网络层级的输出结果，对 YOLO 有了更深刻的了解。

严格说起来，这是我的第一个 Github 项目：walktree/libtorch-yolov3
